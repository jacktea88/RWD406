# 重點整理:

## 條紋表格
.table-striped 類,將在<tbody> 内的行上看到條紋

## 帶邊框表格
.table-bordered 類可以為表格加入邊框

## 滑鼠滑過狀態表格
.table-hover 類可以為表格的每一行加入滑鼠滑過效果(灰色背景)

## 黑色背景表格
.table-dark 類可以為表格加入黑色背景

## 黑色條紋表格
使用 table-dark和 table-striped 類可以建立黑色的條紋表格

## 滑鼠滑過效果-黑色背景表格
使用 table-dark 和 .table-hover 類可以設置黑色背景表格的滑鼠懸停效果

## 表頭顏色
Bootstrap v5中.thead-dark 類用於給表頭加入黑色背景, thead-light 類用於給表頭加入灰色背景

## 較小的表格
.table-sm 類用於通過減少內邊距來設置較小的表格

## 響應式表格
.table-responsive 類用於建立響應式表格:在螢幕寬度小於992px 時會建立水平捲軸,如果可視區域寬度大於992px 則沒有捲軸

以下類設定在指定螢幕寬度下顯示捲軸:
類名                    螢幕寬度
.table-responsive-sm = 装置寛度<576px
.table-responsive-md = 装置寛度<768px
.table-responsive-lg = 装置寬度<992px
.table-responsive-xl = 装寛寬度<1200px

## 表格顏色類
.table-primary:藍色。
.table-success:綠色。
.table-info:淺藍色。
.table-warning:橘色。
.table-danger :紅色ㆍ
.table-secondary:灰色。
.table-dark:深灰色。
